アプリ紹介

「Cookie !!」は、誰でも簡単に楽しめる、中毒性抜群の本格クリッカーゲームです。
最初はたった1枚のクッキーから。あなたの指先ひとつで、宇宙規模のクッキー帝国を築き上げましょう！

🍪 シンプル操作

大きなクッキーをタップするだけ！焼いたクッキーで建物を建てて生産力を爆発させよう。

✨ フィーバータイム

フィーバースターで生産力7倍！さらに激レアな「レインボースター」で大量ボーナスのチャンス。

💎 ジュエルで進化

ジュエルを使って「爆速クリッカー」や「幸運の星」を解放。インフレを加速させよう！

🌟 お菓子図鑑

レアなお菓子を見つけて図鑑をコンプリート。全種類見つけられるかな？

ゲームの特徴

  • インフレの快感: おばあちゃん、工場、そしてクッキー惑星へ……生産力が桁違いに増えていく快感！
  • 放置でも増える: アプリを閉じている間も、建物たちがクッキーを焼き続けます。
  • やり込み要素: 多彩なアップグレードアイテムで、自分だけの最強の生産ラインを構築。
